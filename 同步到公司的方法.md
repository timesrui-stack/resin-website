# 如何在公司继续编辑网站

## 方案1：使用Git同步（推荐⭐）

### 步骤：
1. **在家里电脑上传到GitHub**
   ```bash
   # 在 resin-website 文件夹打开命令行
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/resin-website.git
   git push -u origin main
   ```

2. **在公司电脑下载**
   ```bash
   git clone https://github.com/你的用户名/resin-website.git
   ```

3. **每次修改后同步**
   ```bash
   # 上传修改
   git add .
   git commit -m "描述修改内容"
   git push

   # 下载最新版本
   git pull
   ```

### 优点：
- ✅ 自动版本管理
- ✅ 可以随时回退
- ✅ 多设备无缝同步
- ✅ 免费

---

## 方案2：使用网盘同步

### 百度网盘/OneDrive/Google Drive
1. 将整个 `resin-website` 文件夹放入网盘同步文件夹
2. 在公司电脑安装同一个网盘客户端
3. 自动同步

### 优点：
- ✅ 简单易用
- ✅ 自动同步

### 缺点：
- ❌ 没有版本控制
- ❌ 可能有同步延迟

---

## 方案3：压缩打包传输

### 步骤：
1. **在家里打包**
   - 右键 `resin-website` 文件夹
   - 选择"发送到" → "压缩（zipped）文件夹"
   - 得到 `resin-website.zip`

2. **传输到公司**
   - 通过微信/QQ/邮件发送给自己
   - 或上传到网盘

3. **在公司解压**
   - 下载 zip 文件
   - 右键 → 解压

### 优点：
- ✅ 最简单
- ✅ 不需要额外工具

### 缺点：
- ❌ 需要手动打包/解压
- ❌ 容易忘记同步最新版本

---

## 方案4：使用GitHub Pages直接托管（推荐⭐⭐）

### 好处：
- 网站直接发布到互联网
- 任何地方都能访问编辑
- 免费域名：`https://你的用户名.github.io/resin-website`

### 步骤：
1. 按方案1上传到GitHub
2. 在GitHub仓库设置中：
   - Settings → Pages
   - Source: Deploy from branch
   - Branch: main → /root → Save
3. 等待几分钟，网站就上线了！

### 在任何地方编辑：
- 可以直接在GitHub网页上编辑文件
- 或者用公司电脑 clone 下来编辑

---

## 推荐组合方案 🌟

**Git + GitHub Pages**
1. 版本控制安全
2. 随时随地编辑
3. 网站直接上线
4. 可以给客户看效果
