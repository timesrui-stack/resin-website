# Timesrui Resin 网站预览版

## 🎨 设计特点

这是一个模仿 Let's Resin 风格的树脂产品电商网站预览版，包含以下特色：

### ✨ 主要功能模块

1. **顶部公告栏** - 显示促销信息和品牌优势
2. **导航栏** - 固定顶部，包含搜索、用户、购物车功能
3. **Hero横幅区** - 渐变背景 + 动画效果
4. **信任徽章** - 4个核心卖点展示
5. **产品分类** - 8大类目，彩色渐变卡片
6. **热销产品** - 6个产品展示，带快速查看功能
7. **Instagram展示区** - 社交媒体作品展示
8. **Newsletter订阅** - 邮件营销收集
9. **页脚** - 完整的导航和公司信息

### 🎯 交互功能

- ✅ 平滑滚动导航
- ✅ 添加购物车动画
- ✅ 通知提示系统
- ✅ 卡片悬停效果
- ✅ 滚动加载动画
- ✅ 响应式设计（支持手机端）

### 🎨 设计风格

- 渐变色主题（紫色系）
- 现代化圆角设计
- 阴影和动画效果
- 清晰的视觉层级
- 类似 Let's Resin 的布局结构

## 📂 文件结构

resin-website/

## 🚀 如何使用

1. 直接双击 `index.html` 在浏览器中打开
2. 或使用本地服务器运行（推荐）

## 🔧 定制说明

### 修改品牌颜色
在 `style.css` 中搜索 `#667eea` 和 `#764ba2` 替换为你的品牌色

### 修改Logo
在 `index.html` 第29行修改 `TIMESRUI RESIN` 文字或添加图片

### 添加产品
复制 `.product-card` 代码块并修改内容
