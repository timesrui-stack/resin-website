### 2. 修改颜色

**位置：** `style.css`

#### 当前配色方案：
- 主色：粉红色 `#ff6b9d`
- 辅助色：浅粉 `#ffa8c5`

#### 修改方法：

**方法1：全局替换（推荐）**

在 VS Code 中：
1. 按 `Ctrl+H`（查找替换）
2. 查找：`#ff6b9d`
3. 替换为：你想要的颜色（如蓝色 `#4a90e2`）
4. 点击"全部替换"

**方法2：单独修改**

找到具体位置修改，例如：

```css
/* 修改按钮颜色 */
.btn-primary {
    background: linear-gradient(135deg, #ff6b9d 0%, #ffa8c5 100%);
}

/* 改成蓝色 */
.btn-primary {
    background: linear-gradient(135deg, #4a90e2 0%, #64b5f6 100%);
}
```

#### 常用颜色代码参考：

| 颜色 | 代码 |
|------|------|
| 粉红色 | `#ff6b9d` |
| 蓝色 | `#4a90e2` |
| 绿色 | `#2d7a3e` |
| 紫色 | `#9c27b0` |
| 橙色 | `#ff9800` |
| 黑色 | `#333333` |
| 白色 | `#ffffff` |

### 4. 更换产品图片

**位置：** `index.html`

#### 步骤1：准备图片
1. 把产品图片放到 `resin-website` 文件夹
2. 建议命名：`product-1.jpg`, `product-2.jpg` 等
3. 推荐尺寸：800x800 像素

#### 步骤2：修改代码

**原来（渐变色背景）：**
```html
<div class="product-image" style="background: linear-gradient(135deg, #ffeaa7 0%, #fab1a0 100%);">
    <span style="font-size: 64px;">💎</span>
</div>
```

**改成真实图片：**
```html
<div class="product-image">
    <img src="product-1.jpg" alt="产品名称" style="width:100%; height:100%; object-fit:cover;">
</div>
```

## 🔄 修改流程（重要！）

### 标准流程：

#### 第1步：备份文件 ⚠️

1. 找到文件夹：`C:\Users\apple\.easyclaw\workspace\resin-website`

## ❓ 常见问题

### Q1: 修改后刷新浏览器看不到变化？

**解决方法：**
- 按 `Ctrl+F5` 强制刷新（清除缓存）
- 或者按 `Ctrl+Shift+Delete` 清除浏览器缓存

### Q2: 修改后页面显示乱了？

**原因：** 可能不小心删除了重要代码

**解决方法：**
1. 从备份恢复文件
2. 仔细检查修改的地方，确保没有删除 `<` `>` `{` `}` 等符号

### Q3: 中文显示乱码？

**解决方法：**

在 `index.html` 顶部确认有这行代码：
```html
<meta charset="UTF-8">
```

用记事本保存时选择 "UTF-8" 编码。

### Q4: 图片不显示？

**检查项：**
1. 图片文件是否在 `resin-website` 文件夹里？
2. 文件名是否写对了？（区分大小写）
3. 文件格式是否支持？（推荐 `.jpg` `.png`）

**示例：**
```html
<!-- 错误写法 -->
<img src="Product-1.jpg">  <!-- 文件名实际是 product-1.jpg -->

<!-- 正确写法 -->
<img src="product-1.jpg">
```

### Q5: 怎么改成中文网站？

直接修改所有英文文字为中文即可，例如：

```html
<!-- 英文版 -->
<h2>Shop by Category</h2>

<!-- 中文版 -->
<h2>产品分类</h2>
```

### Q6: 怎么添加新页面（如"关于我们"）？

1. 复制 `index.html`
2. 重命名为 `about.html`
3. 修改内容
4. 在导航栏添加链接：

```html
<li><a href="about.html">关于我们</a></li>
```

## 🚀 进阶技巧

### 1. 修改移动端显示

在 `style.css` 最底部有响应式设计代码：

```css
@media (max-width: 768px) {
    /* 这里的代码只在手机上生效 */
    .hero-title {
        font-size: 28px;  /* 手机上标题更小 */
    }
}
```

### 2. 添加动画效果

已有的动画在 `script.js` 中，可以调整速度：

```javascript
// 找到这段代码，修改时间（单位：毫秒）
setTimeout(() => {
    notification.remove();
}, 3000);  // 改成 5000 就是5秒后消失
```

### 3. 修改字体

在 `style.css` 顶部：

```css
body {
    font-family: 'Poppins', sans-serif;
}
```

改成中文友好字体：

```css
body {
    font-family: 'Microsoft YaHei', '微软雅黑', sans-serif;
}
```
