✏️ 常见修改操作

1. 修改文字内容

位置： index.html

示例1：修改首页大标题

找到这段代码：

<h2 class="hero-title">✨ Create Your Own Magic ✨</h2>
<p class="hero-subtitle">Premium DIY resin supplies for crafters, artists & dreamers 💕</p>

修改成你想要的：

<h2 class="hero-title">✨ 创造你的专属艺术 ✨</h2>
<p class="hero-subtitle">专业树脂材料，让创意触手可及 💕</p>

2. 修改颜色

位置： style.css

当前配色方案

  • 主色：粉红色 #ff6b9d
  • 辅助色：浅粉 #ffa8c5

方法1：全局替换（推荐）

在 VS Code 中：

  1. 按 Ctrl+H（查找替换）
  2. 查找：#ff6b9d
  3. 替换为：你想要的颜色（如蓝色 #4a90e2）
  4. 点击"全部替换"

常用颜色代码参考

颜色 代码

蓝色 #4a90e2
绿色 #2d7a3e
紫色 #9c27b0
橙色 #ff9800

3. 修改产品价格

找到：

<div class="product-price">
    <span class="price">$29.99</span>
    <span class="original-price">$39.99</span>
</div>

修改成：

<div class="product-price">
    <span class="price">¥199</span>
    <span class="original-price">¥299</span>
</div>

4. 更换产品图片

步骤1：准备图片

  1. 把产品图片放到 resin-website 文件夹
  2. 建议命名：product-1.jpg, product-2.jpg 等
  3. 推荐尺寸：800x800 像素

步骤2：修改代码

原来（渐变色背景）：

<div class="product-image" style="background: linear-gradient(...);">
    <span style="font-size: 64px;">💎</span>
</div>

改成真实图片：

<div class="product-image">
    <img src="product-1.jpg" alt="产品名称" 
         style="width:100%; height:100%; object-fit:cover;">
</div>

5. 修改Logo

方法1：替换图片文件

  1. 准备新logo图片（PNG格式，透明背景最佳）
  2. 重命名为 logo-original.png
  3. 替换掉原来的文件

调整Logo大小

在 style.css 中找到：

.logo-img {
    height: 60px;
    width: auto;
}

修改高度：

.logo-img {
    height: 80px;  /* 改成你想要的高度 */
    width: auto;
}

🔄 修改流程（重要！）

  1. 删除当前的 resin-website 文件夹
  2. 把备份的文件夹重命名回 resin-website
  3. 重新开始修改

❓ 常见问题

Q1: 修改后刷新浏览器看不到变化？

解决方法：
  • 按 Ctrl+F5 强制刷新（清除缓存）
  • 或者按 Ctrl+Shift+Delete 清除浏览器缓存

Q2: 修改后页面显示乱了？

原因： 可能不小心删除了重要代码

解决方法：
  1. 从备份恢复文件
  2. 仔细检查修改的地方，确保没有删除 < > { } 等符号

Q3: 中文显示乱码？

解决方法：

在 index.html 顶部确认有这行代码：

<meta charset="UTF-8">

用记事本保存时选择 "UTF-8" 编码。

Q4: 图片不显示？

检查项：
  1. 图片文件是否在 resin-website 文件夹里？
  2. 文件名是否写对了？（区分大小写）
  3. 文件格式是否支持？（推荐 .jpg .png）
